# macos-lock

> Lock your Mac from the command-line


## Install

[Download](https://github.com/sindresorhus/macos-lock/releases/latest) the binary and put it in `/usr/local/bin`.


## Usage

```
$ lock
```


## Build

```
$ ./build
```
